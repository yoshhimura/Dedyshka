Требования к системе "Умный список задач"
1. Описание системы
Система "Умный список задач" — это приложение, позволяющее пользователям эффективно управлять своими задачами. Она предоставляет инструменты для создания, редактирования и удаления задач, а также для их организации по приоритетам, категориям, дедлайнам и напоминаниям. Система включает функции поиска и фильтрации, что позволяет быстро находить нужные задачи и просматривать их в удобном для пользователя виде.

2. Функциональные требования

2.1. Управление задачами
Создание задачи: Пользователь может создать новую задачу, указав название, описание, категорию, приоритет и дедлайн.
Редактирование задачи: Пользователь может изменять любые поля существующей задачи (название, описание, категорию, приоритет, дедлайн).
Удаление задачи: Пользователь может удалить задачу из списка. Удаление должно подтверждаться для предотвращения случайных действий.

2.2. Приоритеты задач
Задачи могут иметь один из трех уровней приоритета: низкий, средний, высокий.
Приоритет отображается визуально (например, цветом или значком) в списке задач.
Пользователь может сортировать задачи по приоритету.

2.3. Категории задач
Пользователь может создавать, редактировать и удалять категории задач (например: "Работа", "Личное", "Учеба").
Каждая задача должна быть привязана к одной категории.
Пользователь может фильтровать задачи по категориям.

2.4. Поиск и фильтрация
Поиск: Пользователь может искать задачи по названию, описанию или категории.
Фильтрация: Пользователь может фильтровать задачи по:
Приоритету (низкий/средний/высокий)
Категории
Статусу (все/активные/завершенные)
Дедлайну (ближайшие/просроченные/все)

2.5. Дедлайны и напоминания
Каждая задача может иметь дедлайн (дата и время).
Пользователь может включать/выключать напоминания для задачи.
Напоминания отправляются пользователю заранее (настраиваемое время до дедлайна) через push-уведомления или email.
Задачи с просроченным дедлайном выделяются в интерфейсе.

3. User Stories
Как пользователь, я хочу создавать новые задачи с указанием названия, описания, категории и дедлайна, чтобы структурировать свои дела и не забывать о важных делах.
Как пользователь, я хочу фильтровать задачи по приоритету и категории, чтобы быстро находить самые важные или относящиеся к определенной сфере задачи.
Как пользователь, я хочу получать напоминания о задачах за 1 час до дедлайна, чтобы успевать выполнить задачи в срок и не пропускать важные события.
